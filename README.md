### **전체 아키텍처**

`Naver 검색 API (뉴스/블로그/쇼핑/이미지/웹/카페)
       │
       ▼
   MCP 도구 호출 (naver_search / naver_search_stream)
       │
       ▼
   Kafka Topic: naver-search-results (produce)
       │
       ▼
   consume_naver_search (consume + 필터링)`

### **추가된 MCP 도구 (6개)**

| **#** | **도구명** | **설명** |
| --- | --- | --- |
| 8 | `naver_datalab_trend` | 데이터랩 검색어 트렌드 → Kafka *(API 권한 추가 후 사용 가능)* |
| 9 | `naver_datalab_shopping` | 데이터랩 쇼핑인사이트 → Kafka *(API 권한 추가 후 사용 가능)* |
| 10 | `consume_naver_trends` | 데이터랩 트렌드 Kafka consume |
| **11** | **`naver_search`** | **검색 API(뉴스/블로그/쇼핑 등) 호출 → Kafka produce** ✅ |
| **12** | **`naver_search_stream`** | **다건 페이징 수집(최대 500건) → Kafka 일괄 produce** ✅ |
| **13** | **`consume_naver_search`** | **Kafka에서 검색 데이터 consume (쿼리/유형 필터링 지원)** ✅ |

### **Kafka 토픽 (3개 생성됨)**

- `naver-datalab-trends` - 데이터랩 트렌드 데이터
- `naver-datalab-shopping` - 데이터랩 쇼핑 데이터
- `naver-search-results` - 검색 API 결과 데이터 ✅ 테스트 완료

### **테스트 결과**

- ✅ Naver 뉴스 검색 API → Kafka produce → consume 정상
- ✅ Naver 블로그 검색 API → Kafka produce 정상
- ✅ TypeScript 빌드 성공
